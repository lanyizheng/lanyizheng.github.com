#ubuntu 环境下，安装jekyll的步骤
- 安装ruby

```
sudo apt-get install ruby1.9.1
sudo apt-get install ruby1.9.1-dev
```

- 安装markdown解析器

```
gem install rdiscount
```

- 安装Pygments语法高亮

```
sudo apt-get install python-pygments
```

- 安装jekyll

```
gem install jekyll
```

-  nodejs

```
sudo apt-get install nodejs
```
